== Description ==
 
### About plugin

The __ZooSpas__ plugin allows you to display animal cards with the option to use filters to sort and select animals for adoption. 
You can keep and display animal records in the best for visitors way. Additional features are settings for two CTA buttons and feedback form.

Задача плагина – облегчить посетителям сайта поиск и выбор животных, которых можно забрать из приюта. 
Администратор сайта может вывести и настроить две универсальные кнопки с целевыми действиями. 

В плагине предусмотрена форма обратной связи с различными типами отображения, благодаря чему она превращается в универсальный инструмент общения с пользователями и получения обратной связи.

Плагин легко устанавливается и требует минимум настроек.

Плагин предназначен для всех, кто занимается помощью и пристройством бездомных животных — некоммерческие организации, общественные объединения, физические лица.

### Основные функции

* Создание и вывод на страницы карточки животного
* Выбор из трёх типов карточек: кошки, собаки, другие животные
* Поиск животных через фильтр
* Стилизация фильтров
* Две универсальные кнопки целевых действий
* Форма обратной связи с различными типами отображения
